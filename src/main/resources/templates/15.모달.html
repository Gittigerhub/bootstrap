모달(대화상자)-창(부모), 대화상자(자식)으로 구성
        대화상자를 닫기전에 부모창으로 이동불가능
        입력폼, 수정폼

        [대화상자를 동작시킬 버튼]
        data-bs-toggle="modal"
        data-bs-target="#아이디" : 해당 아이디의 내용을 대화상자로 표시

        [대화상자폼] - 가능하면 HTML에 하단에 작업
        div .modal id="아이디" : 해당영역인 대화상자 영역
        div .modal-dialog : 대화상자
            .modal-content : 내용구성
            .modal-header
            .modal-body
            .modal-footer
        container > row > col > 요소구성

        [대화상자]
        단점 : 대화상자는 이쁘지만 모든페이지를 대화상자 한페이지로 작업하기 때문에 너무 코드가 길어진다.
        장점 : 이쁘다
